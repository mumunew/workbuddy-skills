# WorkBuddy Skills

开源 WorkBuddy AI助手技能库。

## naval-decision（纳瓦尔人生决策助手）

基于《纳瓦尔宝典》核心思想的人生决策框架。

**功能：**
- 5维度评分系统（专长/杠杆/复利/市场/可持续）
- 4条决策规则（痛苦优先/无杠杆降级/拒绝焦虑/闭环优先）
- 3种状态识别（行动态/接受态/内耗态）
- 结构化输出：评分表+推荐+原因+行动

**触发词：** 纠结、选哪个、怎么选、帮我决定、纳瓦尔、人生决策、该不该做

## 安装方式

将 Skill 文件夹放入 WorkBuddy 的 ~/.workbuddy/skills/ 目录下即可自动加载。
